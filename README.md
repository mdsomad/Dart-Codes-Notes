
# Dart language 


removeAt
Removes the object at position index from this list.


### List removeAt() Method explained

```sh
  
   List<int> numList = [1,2,3,4,5,6,7,8];
  
  //* provide Remove element index 
  numList.removeAt(3);
 
  
  print(numList); 

  
  //* 4 elements removed
  //* numList Result this --> [1, 2, 3, 5, 6, 7, 8]

  ```
